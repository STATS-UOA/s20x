panel.hist(s20x)R Documentation

A Panel Function for Plotting Histograms

Description

This function is called by pairs20x to draw histograms in panels.

Usage

panel.hist (x, ...)

Arguments

x A vector of observations
... Additional arguments, parsed to "rect" which draws the histogram bars.

See Also

"pairs", "pairs20x", "panel.smooth", "panel.cor"
